# Simple Data Cleaning Agent 🤖🧹

This is a beginner-level AI agent that cleans any messy dataset.

## What Does This Agent Do?
- Loads the dataset
- Finds missing values and fixes them
- Removes duplicate rows
- Saves a clean version of the dataset

## Who Made This?
This project was built by a 10-year-old beginner to learn AI and Python.

## Tools Used
- Python
- Pandas
- Kaggle Notebook

## How to Run
1. Open the Kaggle notebook
2. Upload the dataset
3. Run the notebook
4. Download the cleaned dataset
